StepLy
Primary Project · Location Intelligence Platform
Problem
Urban navigation and location discovery lack contextual intelligence about crowd dynamics, temporal patterns, and real suitability — people rely on static ratings instead of living spatial signals.
Solution
A location recommendation and crowd intelligence system that leverages human mobility data, spatial analytics, and real-time signals to deliver context-aware place suggestions adapted to time, crowd density, and user intent.
Architecture
Mobility data pipeline → Spatial feature engineering → ML recommendation model → Crowd intelligence layer → User-facing application
Technical Stack
Python PyTorch Spatial ML Flutter GIS APIs Mobility Data
Vision
Democratize spatial intelligence for everyday navigation — making cities legible through the lens of collective human movement.

EchoMind
Secondary Project · Emotion + Reflection System
Problem
Digital reflection and journaling tools lack emotional intelligence — they capture words but miss the emotional context and patterns that make self-understanding meaningful.
Solution
An emotion-aware reflection system that adapts to user emotional states, identifies patterns over time, and provides intelligent prompts for deeper self-understanding.
Vision
Bridge the gap between digital interaction and emotional awareness — making technology a partner in personal growth.
